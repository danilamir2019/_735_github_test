Программа для выполнения следующих задач:
1. Инициализация указателя стека;
2. Переполнение стека;
3. Передача параметров функций через стек;
4. Использование двух указателей стека.

Для запуска программы необходимо создать проект в IAR Embedded Workbench 6.30.1.3142 (в данной среде разрабатывалась программа) и настроить его в соответствии с документацией по настройке IAR с сайта Миландр, заисключением того, что отладчик устанавливаем в режим «симуляции». После в созданный проект добавляем файл «main.c».
Для выполнения нужного участка кода необходимо раскомментировать соответствующий #define, другие же #define должны оставаться закомменченными, если нет нужды в выполнении участка кода с соответствующим наименованием.
